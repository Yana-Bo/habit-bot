# HabitBot

HabitBot — учебный Telegram-бот для отслеживания привычек.

Проект разработан в рамках практических работ по дисциплине 
«Разработка и тестирование программных систем».

## Возможности

Бот поддерживает базовый функционал для работы с привычками:

- создание привычки
- просмотр списка привычек
- отметка привычки как выполненной
- просмотр статистики выполнения
- удаление привычки

## Команды бота

- `/start` — запуск бота и справка  
- `/add <название>` — добавить привычку  
- `/list` — показать список привычек  
- `/done <id>` — отметить привычку выполненной  
- `/stats <id>` — посмотреть статистику привычки  
- `/delete <id>` — удалить привычку  

## Структура проекта

- `habits.py` — основная логика работы с привычками  
- `tests/` — модульные тесты  
- `.github/workflows/` — CI для автоматического запуска тестов  
- `README.md` — описание проекта  

## Технологии

- Python
- python-telegram-bot
- GitHub Actions

## Статус проекта

Учебный проект. Бот создан для демонстрации интеграции с Telegram API 
и не предназначен для промышленного использования.
